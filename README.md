# ITA智创协会 (InTech Association) 官方网站

这是ITA智创协会的官方网站，一个展示社团活动、宗旨和精神的静态网页。

## 功能特点

- 🌐 **中英双语支持**：点击右上角的语言切换按钮即可在中英文之间切换
- 🎠 **轮播图展示**：首页包含5张轮播图片，展示社团风采
- 📱 **响应式设计**：适配各种设备屏幕尺寸
- 🎨 **美观的UI设计**：现代化的界面设计，舒适的视觉体验
- ⚡ **平滑滚动**：导航栏链接点击后平滑滚动到对应部分

## 文件结构

```
ClubWebpage/
├── index.html          # 主页面文件
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── main.js         # JavaScript功能文件
├── images/
│   └── WIP.png         # 占位图片（可替换为实际图片）
└── README.md           # 项目说明文件
```

## 使用说明

1. 直接在浏览器中打开 `index.html` 文件即可查看网站
2. 如需替换图片，请将新的图片放入 `images` 文件夹，并修改 `index.html` 中相应的图片路径
3. 如需修改网站内容，请编辑 `index.html` 文件中的文本内容
